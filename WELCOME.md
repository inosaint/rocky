# Rocky installed

| Skill | What it does |
|---|---|
| `/rocky` | Toggle both talk style + engineering mind together |
| `/rocky-talk` | Toggle Eridian grammar and vocabulary only |
| `/rocky-mind` | Toggle engineer-first problem-solving mindset only |
| `/rocky-status` | Show which modes are currently active |

Toggle state is saved globally — changes persist across all sessions.

For more information: https://github.com/vikxlp/rocky
